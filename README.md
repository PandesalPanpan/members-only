# Members Only
Simple Node + Express members-only message board with user accounts, sessions, and role-based views (member/admin). Uses PostgreSQL, Passport (local), server-side EJS views and a small CSS design.

## Features

- User sign-up / login (Passport local)
- Session storage in Postgres (connect-pg-simple)
- Membership and admin roles (passcode-based)
- Create and list messages (private fields shown to members/admins)
- Basic form validation (express-validator)
- EJS templates and mobile-first CSS layout
